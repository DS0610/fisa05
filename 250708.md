### 정규 표현식
```
import re

p = re.compile(r"happy", re.I) # re.I -> 대소문자 무시
p = re.compile(r"happy", re.M) # multiline

re.sub() # 대체할 때 사용

p = re.compile(r"?P<패턴명>패턴")
p.sub(r"\g<패턴명>")
```

### f-string
```
a = 1
b = 2

print(f"The sum of {a} + {b} = {a+b}")

print("The sum of {} + {} = {}".format(a, b, a+b))
```

### % 뒤 문자열 데이터를 출력하는 방법
- %d - 정수 표현
- %s - 문자 표현
- %f - 소수 표현
